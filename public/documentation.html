Movie API Documentation

Welcome to the Movie API! This API allows you to retrieve information about various movies. You can search for movies by title, get details about specific movies, and more.

Request URL Response
GET /movies JSON array of all movies
GET /movies/:title JSON object with movie details (description, genre, director, image URL, featured)
GET /genres/:name JSON object with genre description
GET /directors/:name JSON object with director's bio, birth year, death year
POST /users JSON object with new user details
PUT /users/:userId JSON object with updated user details
POST /users/:userId/favorites Text indicating the movie has been added to favorites
DELETE /users/:userId/favorites/:movieId Text indicating the movie has been removed from favorites
DELETE /users/:userId Text indicating the user's email has been removed
